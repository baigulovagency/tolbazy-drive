# Толбазы Драйв

Telegram-платформа для заказа такси в с. Толбазы и Аургазинском районе.

## Что уже заложено

- 3 стартовых автопарка: Вояж, Александр, Акбузат.
- Возможность добавлять новые автопарки.
- 10 тарифных зон: 5 зон Толбаз и 5 зон деревень района.
- Стартовые тарифы: 150 ₽ по Толбазам, 300 ₽ для деревень.
- Race-модель: заказ приходит всем свободным водителям, кто первый принял - тот забрал.
- Пассажир, водитель, диспетчер парка, админ парка, суперадмин.
- Telegram bot, Fastify API, PostgreSQL, Prisma, Next.js админка.

## Быстрый старт

1. Скопируйте переменные окружения:

```bash
cp .env.example .env
```

2. Запустите PostgreSQL:

```bash
docker compose up -d
```

3. Установите зависимости:

```bash
npm install
```

4. Создайте таблицы и наполните стартовыми зонами/тарифами:

```bash
npm run db:migrate
npm run db:seed
```

5. Запустите API и админку:

```bash
npm run dev
```

API: `http://localhost:4000`

Админка: `http://localhost:3000`

## Важное

В текущем окружении `git` и команды Node не удалось проверить через терминал. Если они не установлены, сначала установите Git, Node.js LTS и Docker Desktop.
